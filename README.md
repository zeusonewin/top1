# The Dog House — лендинг

Статичный одностраничный сайт про слот The Dog House (Pragmatic Play).

## Структура проекта

```
├── index.html          — главная страница (открывается по умолчанию)
├── favicon.svg         — иконка сайта (тёмно-зелёная тема)
├── favicon.jpg         — опционально: растровая иконка (положите в корень при необходимости)
├── images/
│   └── og-image.jpg    — картинка для соцсетей 1200×630 (добавьте вручную)
└── README.md
```

Файл с длинным именем `The Dog House слот играть онлайн — демо собак от Pragmatic Play бесплатно.html` — копия главной страницы для SEO/архива; для деплоя используется `index.html`.

## Как открыть локально

1. Дважды кликнуть по `index.html` в проводнике, или  
2. Запустить локальный сервер из папки проекта, например:
   - `npx serve .`
   - или открыть папку в VS Code и использовать расширение Live Server

## Деплой

Папку можно залить на любой хостинг статики (Vercel, Netlify, GitHub Pages и т.п.).  
Убедитесь, что в корне доступны `index.html` и `favicon.svg`; при необходимости добавьте `favicon.jpg` и `images/og-image.jpg`.
