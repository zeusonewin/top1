Картинки для сайта The Dog House
================================

og-image.jpg
------------
Положите сюда файл og-image.jpg для превью в соцсетях (Open Graph, Twitter).
Рекомендуемый размер: 1200×630 px.
Ссылка в коде: https://the-dog-house.versel.app/images/og-image.jpg

favicon.jpg (в корне проекта)
-----------------------------
По желанию можно добавить favicon.jpg в корень папки — тогда он будет использоваться
как иконка там, где браузер не поддерживает SVG (например, часть старых устройств).
Сейчас используется favicon.svg из корня.
